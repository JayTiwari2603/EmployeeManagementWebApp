# Employee CRUD Spring + React 🚀

A simple and full-stack **Employee Management WebApp** built with:

✅ **Spring Boot (Java + Maven)** as the backend  
✅ **React + Tailwind CSS** as the frontend  
✅ **RESTful APIs** for communication  
✅ **CRUD operations**: Create, Read, Update, Delete

## ▶️ How to Run the Backend (Spring Boot)

cd C:\Users\hp\Desktop\JavaCRUDProject

# Using Maven Wrapper (Windows)
.\mvnw.cmd spring-boot:run 

# OR using Maven (if installed)
mvn spring-boot:run

✅ Server runs on: http://localhost:8080

▶️ How to Run the Frontend (React)
cd C:\Users\hp\Desktop\JavaCRUDProject\crud-project-view
npm install
npm start

✅ App opens at: http://localhost:3000

🔗 API Endpoints
Method	Endpoint	Description
GET	/employees	Get all employees
GET	/employees/{id}	Get employee by ID
POST	/employees	Add new employee
PUT	/employees/{id}	Update employee
DELETE	/employees/{id}	Delete employee
